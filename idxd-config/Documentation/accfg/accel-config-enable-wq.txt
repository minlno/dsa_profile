accel-config enable-wq(1)
=========================

NAME
----
accel-config-enable-wq - enables an accelerator device work queue

SYNOPSIS
--------
[verse]
'accel-config enable-wq' <device_name>/<workqueue_name>

EXAMPLE
-------
accel-config enable-wq dsa0/wq0.0
